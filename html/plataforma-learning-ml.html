¿Cómo funciona Learning ML?

Learning ML funciona por etapas.

 Para que cualquier modelo funcione, debemos pasar por estas tres fases, ¡como si estuviéramos entrenando a una mascota!

¿Qué tipos de ejercicios se puede crear con la plataforma Learning ML?

 En la plataforma Learning ML, podemos crear tres tipos de inteligencias o modelos (Cerebros de robot) según lo que queramos que el robot aprenda: 

Reconocimiento de textos: Reconocimiento numérico: Reconocimiento de imágenes:
El robot aprende a leer palabras o frases y las pone en grupos. El robot usa los números para entender cosas. El robot usa la cámara o fotos para reconocer objetos.

Empecemos con los ejercicios

Una vez ingresado en la versión básica de la plataforma se redirigirá al editor de modelos.

  
 
Damos clic y seleccionamos, ya sea para texto o imágenes.
 
 

Una vez dentro de la plataforma procedemos a crear tanto texto como imágenes, seleccionamos una función para crear.


 

Reconocimiento de Texto

El reconocimiento de textos es una función que permite al modelo identificar y clasificar palabras o frases escritas:


Seleccionamos para crear categorías o apartados, para poder añadir los elementos que debe analizar la IA.




 

Una vez creado las dos o más categorías, continuamos con los contenidos que debe o va a llevar cada apartado.
Pasamos al punto 2. Aprender, y seleccionamos aprender a reconocer textos y esperamos a que finalice para pasar al siguiente punto.

Ponemos a prueba nuestra IA y verificamos si funciona con las categorías dadas.

Reconocimiento de Imágenes

 
 El reconocimiento de imágenes es lo mismo que con el texto solo que en vez de texto se usan imágenes.




Seleccionamos para crear categorías o apartados, para poder añadir las imágenes que debe analizar la IA. 


Una vez creado las dos o más categorías, continuamos con las imágenes que debe o va a llevar cada apartado.

Pasamos al punto 2. Aprender, y seleccionamos aprender a reconocer imágenes y esperamos a que finalice para pasar al siguiente punto.

Ponemos a prueba nuestra IA y verificamos si funciona con las categorías dadas.
